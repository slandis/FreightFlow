# FreightFlow

A web-based warehouse and logistics simulation game built with Phaser, React, TypeScript, and Vite.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run test`

## Project Notes

The current implementation is a starter scaffold. The design and architecture source documents live in `memory-bank/`.

## Development Phases

Please see `memory-bank/implementation-plan.md` for detailed phase plans, and `memory-bank/progress.md` for a quick rundown of any completed phases (current complete through Phase 3.)
